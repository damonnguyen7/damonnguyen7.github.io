Califight

A social media platform for fighters allowing them to create and manage local events. I've really wanted to create a platform to help fighters get more competition experience without having to spend a lot of money on over priced tournaments by allowing a community to create local events and a platform to be able to find fighters and events.

Tech Stack

  • React
  • Redux
  • Node
  • Express
  • DynamoDB
  • AWS
